# viSQL
Scan SQL vulnerability on target site and sites of on server.

# Installation

```
~$ git clone https://github.com/blackvkng/viSQL.git
~$ cd viSQL
~# python2 -m pip install -r requirements.txt
~$ python2 viSQL.py --help
```
